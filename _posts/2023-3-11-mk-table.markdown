### 生成 markdown 目录
在 vscode 中安装 `Markdown All in One` 插件。文章写完后，按下快捷键 `command + shift + p`，输入`Markdown All in One: Create Table of Contents` 会自动生成文章的目录。

如果文章目录有更新，目录会自动更新。

`![java-javascript](/pics/somepic.png)`

### mac快捷键
* 搜索文件名：快捷键为 `command + p`
* 全局搜索：`command + shift + f`
